## ⚡ 3 Passos para começar

### 1️⃣ Instalar
```bash
cd frontend
npm install
```

### 2️⃣ Configurar
```bash
# Criar arquivo: frontend/.env.local
VITE_API_URL=http://localhost:3000/api
```

### 3️⃣ Rodar
```bash
# Terminal 1
cd backend && npm start

# Terminal 2
cd frontend && npm run dev

# Abrir: http://localhost:5173/admin/login
```
