                        COMO COMECCAR
================================================================================

1. npm install

2. Configurar .env.local
   VITE_API_URL=http://localhost:3000/api

3. Rodar backend (Terminal 1)
   cd backend
   npm start

4. Rodar frontend (Terminal 2)
   cd frontend
   npm run dev

5. Acessar
   http://localhost:5173/admin/login

6. Login
   username: admin
   password: admin123

7. Pronto!
